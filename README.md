8Store Apps — База данных приложений
Этот репозиторий является удаленным сервером для магазина 8Store. Здесь хранится файл конфигурации apps.json и сами установочные пакеты приложений.

Как добавить приложение в магазин:
Загрузите ваш .appx файл в этот репозиторий.

Обновите файл apps.json, добавив новый блок данных.

Шаблон для добавления в apps.json:
JSON

{
  "ID": "Уникальный_номер",
  "Title": "Название приложения",
  "Category": "Категория",
  "IconPath": "none",
  "DownloadUrl": "прямая_ссылка_на_raw_файл",
  "MinOSVersion": "6.2",
  "MaxOSVersion": "6.3"
}
